# The-Art-of-Prompting

This repo contains a creative framework, based on Chinese metaphysics, for prompt engineering system messages for large language model based agents. 

## Concept

In this framework, every system message consists of these six layers:
1. Identity Declaration
2. Contemplation Structure
3. Core Themes
4. Expression Methods
5. Response Structure
6. Heart Seal

The complete framework is described in The-Art-of-Prompting.md markdown file. Under Contemplation-Methods-Extension.md are additional contemplation modes for the prompts, in Template.md you can find a template for the prompt.
